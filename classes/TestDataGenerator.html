        input?: GenerateRecordsWriteInput,
    ): Promise<GenerateRecordsWriteOutput>

  • Generates a RecordsWrite message for testing. dataBytes & dataStream returned will be undefined as long as dataCid or dataSize is given. Implementation currently uses RecordsWrite.create().

    Parameters

    • Optionalinput: GenerateRecordsWriteInput
      • attesters

        Attesters of the message. Will NOT be generated if not given.

      • data

        Data that belongs to the record. Generated when not given only if dataCid and dataSize are also not given.

      • dataFormat

        Format of the data. Defaults to 'application/json' if not given.

      • signer

        Signer of the message. Generated if not given.

      • schema

        Schema of the message. Randomly generated if not given.
